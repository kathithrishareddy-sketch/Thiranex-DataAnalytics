# Task 2 - Tableau Sales Dashboard

## Objective
Create an interactive Tableau dashboard using sales data.

## Dashboard Includes
- Total Sales KPI
- Total Profit KPI
- Total Orders KPI
- Average Sales KPI
- Monthly Sales Trend
- Sales by Category
- Profit by Region
- Top 10 Products by Sales
- Sales by Segment (Pie Chart)

- CSV Dataset
